# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-06

### Added
- Initial release of Terminal Background Setup Tool
- Support for 6 terminal emulators:
  - Kitty
  - Alacritty
  - Konsole
  - Terminator
  - Cool Retro Term
  - URxvt
- Support for 4 background methods:
  - Static images (PNG, JPG, WebP)
  - Animated GIFs
  - Video backgrounds (MP4, WebM)
  - Live HTML animations
- Pre-built preset themes:
  - Skeleton VHS
  - Matrix Rain
  - Cyberpunk Neon
  - Minimal Blur
- Interactive opacity slider
- File upload functionality
- One-click configuration copy to clipboard
- Responsive design for mobile and desktop
- Beautiful gradient UI with smooth animations
- Installation command display for each terminal
- Comprehensive documentation

### Features
- Tab-based navigation for easy workflow
- Real-time configuration generation
- Compatibility checking between terminals and background methods
- Visual feedback for user actions
- Detailed setup instructions for each terminal
